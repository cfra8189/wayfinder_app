# Executive Summary

**BOX** is a comprehensive SaaS platform designed exclusively for independent music artists and studios to manage their creative assets, protect their intellectual property, and streamline their path from concept to publication.

**Philosophy:** *"In order for creators to be seen out of the box, they have to define their own box."*

BOX provides the tools, structure, and guidance that major labels have always kept behind closed doors - now accessible to independent creators who want to own their destiny.

---

# The Problem We Solve

Independent music artists face critical challenges:

1. **Disorganized Creative Assets** - Songs, recordings, and ideas scattered across devices with no central system
2. **Missing Metadata** - ISRC, UPC, ISWC codes are confusing and often incomplete
3. **IP Vulnerability** - Work released without proper copyright protection
4. **Industry Complexity** - Royalty collection requires submissions to multiple platforms with different formats
5. **Unprofessional Presentation** - No centralized press kit or portfolio for industry opportunities
6. **Studio-Artist Disconnect** - No efficient way for studios to manage and showcase client work

# The BOX Solution

## Complete Feature Set

### 1. Project Tracking System
Track every creative work from initial concept through final publication:
- **Status Workflow**: Concept → Development → Review → Published
- **Project Types**: Singles, EPs, Albums, Beats, Samples, Videos
- **Visual Progress Bar**: See exactly where each project stands in the IP protection journey

### 2. Metadata Management
Store and organize critical industry identifiers:
- **ISRC** (International Standard Recording Code)
- **UPC** (Universal Product Code)
- **ISWC** (International Standard Musical Work Code)
- **Copyright Registration Numbers**
- **PRO Work IDs**
- Writers, Publishers, Release Dates, Duration, Genre, BPM

### 3. IP Protection Workflow
Guided 6-step process to protect creative work:

| Step | Action | Cost |
|------|--------|------|
| 1 | Fix Your Work (record/document) | FREE |
| 2 | Register Copyright (US Copyright Office) | $45-65 |
| 3 | Join a PRO (BMI or ASCAP) | FREE-$50 |
| 4 | Register Composition (get ISWC) | FREE |
| 5 | Upload to Distributor (get ISRC/UPC) | $0-30/yr |
| 6 | Release & Monitor | Ongoing |

### 4. Agreement Generator
Professional music industry agreements ready for download:
- **Split Sheets** - Document ownership percentages between collaborators
- **Basic License** - Non-exclusive licensing for beats and productions
- **Premium License** - Exclusive licensing with advanced terms
- **Production Agreement** - Full production service contracts
- **NDA** - Protect confidential creative discussions

### 5. Creative Space
Private workspace for inspiration and ideation:
- Notes with categories (Inspiration, Lyrics, Ideas, References)
- Media URL embedding for visual/audio references
- Tag system for organization
- Pin important notes to the top
- Drag-and-drop reordering

### 6. Submission File Generator
Export project data in industry-standard formats for:
- **The MLC** (Mechanical Licensing Collective) - Digital mechanical royalties
- **ASCAP/BMI** - Performance royalties via MusicMark EBR
- **Music Reports** - Licensing for TikTok, Amazon Music, etc.
- **SoundExchange** - Digital performance royalties for masters
- **Harry Fox Agency** - Mechanical royalties for reproductions

Each export generates properly formatted CSV files matching each platform's bulk upload requirements.

### 7. Electronic Press Kit (EPK)
Professional press kit builder with:
- **Three Biography Lengths** - Short (elevator pitch), Medium (press), Long (features)
- **Featured Tracks** - Streaming links to best work
- **Photo Gallery** - High-resolution promotional images
- **Video Links** - Music videos and live performances
- **Achievements** - Awards, streaming milestones, chart positions
- **Press Quotes** - Notable reviews and endorsements
- **Contact Information** - Press, booking, and social links
- **Technical Rider** - Equipment needs for venues
- **Stage Plot** - Performance setup requirements

Public EPK accessible via shareable link using unique BOX Code.

### 8. BOX Code System
Unique identifier (BOX-XXXXXX) for networking:
- Share your code to connect with studios and collaborators
- Studios display their code for artist recruitment
- Artists can join studio networks using codes during registration

### 9. Studio Dashboard (Business Accounts)
Tools for studios and labels to manage rosters:
- **Artist Roster Management** - Invite and manage client artists
- **Portfolio Curation** - Feature best work from the roster
- **Public Portfolio Page** - Showcase studio's artists and projects
- **Project Visibility** - View all projects from managed artists

### 10. Community & Sharing
- Submit notes for community visibility (admin approval)
- Public community page with shared inspiration
- Favorites and comments on shared content

### 11. Comprehensive Documentation
Built-in reference guides covering:
- Copyright fundamentals and registration
- PRO membership (ASCAP, BMI, SESAC, GMR)
- Global identifier systems (ISRC, UPC, ISWC)
- Complete IP protection workflow
- Industry terminology and best practices

# User Roles

## Artist Account
Individual music creators who:
- Manage their own projects and creative pipeline

## Studio Account

# Technology Stack

- **Frontend**: React 18 with TypeScript, Vite, Tailwind CSS
- **Backend**: Express.js with TypeScript
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: OAuth (Google, GitHub, Apple, Email) + Email/Password with verification
- **Email**: Resend integration for transactional emails
- **Design**: Terminal aesthetic with light/dark mode, monochrome palette
